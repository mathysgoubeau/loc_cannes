# CTD Conciergerie Premium

Site Next.js ultra minimaliste pour une conciergerie de luxe.

## Lancer

```bash
npm install
npm run dev
```

## Variables

```bash
NEXT_PUBLIC_WHATSAPP_NUMBER=33672208288
NEXT_PUBLIC_SITE_URL=https://votre-domaine.fr
STRIPE_SECRET_KEY=sk_live_xxx
```

Sans clé Stripe, le checkout bascule en simulation 3D Secure avec email et notification interne mockés.
